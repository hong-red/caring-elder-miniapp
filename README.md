# 智享养老微信小程序

## 项目介绍
听暖养老微信小程序是一款专为老年人设计的健康管理和生活服务平台，提供健康监测、用药提醒、家庭关爱等功能，旨在提升老年人的生活质量和健康管理能力。

## 功能特点

### 核心功能
- **健康管理**：记录和监测健康数据，包括血压、血糖、体重等
- **用药提醒**：设置用药时间和剂量，定时提醒服药
- **家庭关爱**：家人绑定功能，实时查看老人健康状态
- **健康咨询**：提供健康知识和咨询服务
- **生活记录**：打卡功能，记录日常活动
- **个人中心**：管理个人信息和系统设置

### 特色功能
- **语音朗读**：点击页面文字即可触发语音朗读，方便视力不好的老年人使用
- **简洁界面**：采用大字体、简洁布局，符合老年人使用习惯
- **安全可靠**：数据加密存储，保护用户隐私

## 技术栈

- **开发框架**：微信小程序原生开发
- **语言**：JavaScript, WXML, WXSS, JSON
- **数据存储**：微信小程序云开发
- **语音服务**：微信小程序语音合成API
- **版本控制**：Git

## 快速开始

### 开发环境搭建
1. 安装微信开发者工具
2. 克隆本项目到本地
3. 在微信开发者工具中导入项目
4. 配置小程序AppID
5. 启动开发服务器

### 项目配置
- 修改`app.json`中的配置项，如页面路径、窗口样式等
- 配置云开发环境

## 使用说明

### 语音朗读功能
1. 在任意页面点击文字内容
2. 系统将自动朗读点击的文字
3. 支持多种语言和语速设置

### 健康数据记录
1. 进入健康页面
2. 点击"添加数据"按钮
3. 选择要记录的数据类型
4. 输入数据并保存

### 用药提醒设置
1. 进入用药页面
2. 点击"添加用药"按钮
3. 填写用药信息，包括药品名称、剂量、时间等
4. 保存设置，系统将在指定时间提醒
